## Lavanderia Automatizada - Trabalho APS

Trabalho feito em dupla: Kamila Rocha e Vitória Barbosa.

Este repositório contém o projeto desenvolvido para o trabalho final da disciplina de **Análise e Projeto de Sistemas**. O objetivo é modelar e implementar um sistema de controle para uma lavanderia, abrangendo desde a documentação das funcionalidades (UML) até a implementação da interface gráfica.

### Sobre o Projeto:

O sistema visa automatizar o processo de recebimento de roupas em uma lavanderia. O fluxo principal consiste em:

1.  O cliente entrega as roupas
2.  O atendente registra o cliente e seleciona os serviços para cada peça (lavar, secar, passar ou serviço completo)
3.  O atendente seleciona o tipo de cada peça (roupa normal, roupa delicada, toalha, roupa de cama etc.)
4.  O software calcula o valor total do serviço com base nos serviços escolhidos e dá um desconto de 10% se o valor total for maior que R$100

### Artefatos:

1.  **Casos de Uso:** Diagrama de casos de uso
2.  **Diagramas de Sequência:** Representação do fluxo de mensagens entre os objetos para os casos de uso
3.  **Diagrama de Classes:** Descrição das classes e como elas se relacionam
4.  **Interface Gráfica:** Tela de Login e Tela de Pedido

### Tecnologias Utilizadas:

- **Linguagens:** HTML, CSS, JAVASCRIPT

- **Modelagem UML:** PlantUML, Lucidchart
